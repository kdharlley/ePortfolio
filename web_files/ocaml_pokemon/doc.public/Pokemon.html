Module Pokemon

module Pokemon: sig .. end

Representation of static pokemon data.

This module represents the data stored in pokemon files, including the pokemon names and description. It handles loading of that data from JSON as well as querying the data.

type pokemon

The abstract type of values representing a pokemon.

The abstract type of the set representing all valid pokemon in the game.

from_json j is the set of pokemon(pokedex) that j represents. Requires: j is a valid JSON pokedex representation.
