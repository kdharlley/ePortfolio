Module Gameworld

module Gameworld: sig .. end

Representation of static game data.

This module represents the data stored in game files, including the room names and description. It handles loading of that data from JSON as well as querying the data.

exception UnknownMovement of string

Raised when an unknown direction is encountered

exception UnBeatenCh of string

Raised when an unbeaten character prevents the game from progressing.

exception DeadEnd

Raised when deadend is encountered

The abstract type of values representing a game

val game_json : Yojson.Basic.t -> Pokemon.t -> game

game_json json pokedex is the game represented by json. pokedex is used to obtain pokemon in the game.

val get_rooms_ext : string -> room list -> room

get_rooms_ext s rms is the room with name s in the list of rooms rms Raises: Failure with an unspecified error message if s is not a valid room name.

val get_room : game -> string -> room

get_room g s is the room with name s in game g Raises: Failure with an unspecified error message if s is not a valid room name. Raises: DeadEnd if s is the empty string.

val get_chars_ext : string -> character list -> character option

get_chars_ext s chrs is None if s is not in chrs, else it is Some chr where chr is the character with name s.

val get_char : room -> string -> character

get_char rm s is the character in room rm with name s Raises: Failure with an unspecified error message if s is not in room rm.

val valid_char : room -> string -> bool

valid_char rm s returns true iff character s is in room r

val get_room_chars : room -> string list

get_room_chars rm is the list of character names in room rm

val start_room : game -> room

start_room game returns the start room of game game

start_room g is the starting room of game g

val char_pokemon : character -> Pokemon.pokemon

char_pokemon ch is the pokemon of character ch

val pokemon_center : room -> bool

pokemon_center r evaluates to true/false depending on if the room has a pokemon center

val north_room : game -> room -> room

north_room g r is the room to the north of room r in game g

val south_room : game -> room -> room

south_room g r is the room to the south of room r in game g

val east_room : game -> room -> room

east_room g r is the room to the east of room r in game g

val west_room : game -> room -> room

west_room g r is the room to the west of room r in game g

val room_desc : game -> string -> string

room_desc g r is the description of the room named r in game g

val room_name : room -> string

room_name r is the name of room r

val room_to_visit : game -> room -> string -> room

room_to_visit g r s is the room to visit in game g when one moves in the direction s in room r Raises: UnknownMovement if s is not a cardinal direction. Raises: DeadEnd if the direction s leads to a dead end.

val elite3 : game -> string list

elite3 g returns the 'elite 3' opponents in game g in the right order

val list_chars : game -> string list

list_chars g returns a list of all character names in game g

val non_elites : game -> string list

non_elites g returns a list of all non-elite character names in game g

val char_diff : game -> string list -> string list

char_diff g ch returns all non-elite character names which are not in ch but are in game g

val get_char_room : string -> game -> string

get_char_room c g returns the room which character c is in in game g. Raises: Failure with an unspecified error message if the character c is not in any room.

val in_elite3 : game -> string -> bool

in_elite3 g ch returns whether the character named ch is in the 'elite 3'

val roster_length : game -> int

roster_length g returns the roster length in game g
